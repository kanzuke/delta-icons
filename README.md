# 🌍 Delta Icons

A lightweight, consistent SVG icon set for natural disasters, designed for direct integration into scripts, dashboards, and web applications.

## 🎯 Purpose

This repository provides a collection of clean, minimal, and scalable SVG icons representing various natural disasters such as earthquakes, tsunamis, wildfires, and more.

The icons are:
- 📦 Easy to integrate (no build tools required)
- 🎨 Fully customizable via CSS (`currentColor`)
- ⚡ Optimized for performance (inline SVG or CDN usage)
- 🧩 Designed with a consistent monoline style

## 🚀 Usage

### 1. Direct CDN usage (recommended)

Use jsDelivr to load icons directly from this repository:
https://cdn.jsdelivr.net/gh/kanzuke/delta-icons@main/icons/earthquake.svg


Example:
<img src="https://cdn.jsdelivr.net/gh/@main/icons/earthquake.svg">

### 2. Raw GitHub URL
https://raw.githubusercontent.com//main/icons/earthquake.svg


### 3. Inline SVG (best for customization)
<svg viewBox="0 0 24 24" stroke="currentColor">
  <!-- SVG content -->
</svg>

## 🎨 Customization

All icons use:

```
stroke="currentColor"
```

This allows dynamic styling:

```css
.icon {
  color: red;
  width: 32px;
  height: 32px;
}
```

---

## 🧠 Design Principles

- Minimal and readable at small sizes
- Consistent stroke width and proportions
- No unnecessary detail
- Works on both light and dark backgrounds
